<h2>📖 Overview</h2>

<p>
The <strong>GreenGuard Intel API</strong> provides real-time
Environmental, Social, and Governance (ESG) intelligence
through a lightweight REST API.
</p>

<p>
It allows developers, researchers, and organizations to integrate ESG
analytics into dashboards, compliance platforms, and sustainability tools.
</p>

<p>
The system is powered by the <strong>GreenGuard AI Engine</strong>,
trained on African ESG datasets to deliver context-aware insights
for businesses, policymakers, and investors.
</p>

<p>
<strong>Dataset:</strong>
<a href="https://huggingface.co/datasets/Allsafeafrica/GreenGuard-Intel-Base">
GreenGuard Intel Base
</a>
</p>

<h2>🔌 API Endpoint</h2>

<p><strong>ESG Insight Query</strong></p>

<pre>
GET /api/esg-insight
</pre>

<p>
Returns ESG intelligence insights based on a user query.
</p>

<h3>Example Request</h3>

<pre>
GET /api/esg-insight?q=climate risk in africa
</pre>

<h3>Example Response</h3>

<pre>
{
 "query": "climate risk in africa",
 "insight": "Climate risks across Africa are increasing due to rising
 temperatures and extreme weather patterns. ESG strategies emphasize
 sustainable infrastructure, climate adaptation, and responsible
 resource management."
}
</pre>

<h2>💻 Running Locally</h2>

<h3>Clone Repository</h3>

<pre>
git clone https://github.com/Allsafeafrica/GreenGuard-Intel-API
cd GreenGuard-Intel-API
</pre>

<h3>Install Dependencies</h3>

<pre>
pip install -r requirements.txt
</pre>

<h3>Run API Server</h3>

<pre>
uvicorn main:app --reload
</pre>

<p>
API will run at:
</p>

<pre>
http://localhost:8000
</pre>

<p>
Interactive documentation:
</p>

<pre>
http://localhost:8000/docs
</pre>
